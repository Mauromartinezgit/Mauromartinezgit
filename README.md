## 🚀 Proyectos

### 🎮 [Juego — Piedra, Papel o Tijera Multijugador](https://juego-pearl.vercel.app)
Juego multijugador en tiempo real con salas sincronizadas, backend propio y deployment separado.

- **Frontend:** TypeScript · HTML5 · CSS3 · Parcel · Vercel
- **Backend:** Node.js · Express · TypeScript · Firebase Firestore · Render
- Salas con códigos únicos · contador de tiempo · sistema de rondas (mejor de 3, al mejor de 5)

🔗 [Ver demo](https://juego-pearl.vercel.app) · [Código frontend](https://github.com/Mauromartinezgit/Juego) · [Código backend](https://github.com/Mauromartinezgit/Juego-backend)

### ✅ [focus-app — To-Do List](https://focus-app-opal.vercel.app)
Aplicación de gestión de tareas con arquitectura basada en componentes y separación de lógica de negocio.

- **Stack:** React · JavaScript · Tailwind CSS · Vite · Vercel
- Custom hooks (`useTasks`, `useLocalStorage`) · drag & drop nativo · design system con CSS variables
- Filtros, estadísticas en tiempo real, barra de progreso animada y responsive mobile first

🔗 [Ver demo](https://focus-app-opal.vercel.app) · [Código](https://github.com/Mauromartinezgit/focus-app)
